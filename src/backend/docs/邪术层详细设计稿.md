Speculative Execution Kernel / StreamShadowEngine v2

版本: Draft v1.0
日期: 2026-04-17
适用范围: Polaris TransactionKernel 内部的流式工具推测执行子系统
目标: 将当前骨架级 StreamShadowEngine 升级为 可控、可取消、可复用、可观测、可灰度发布 的推测执行内核

1. 设计目标
1.1 核心目标

在不改变 Polaris 事务正确性的前提下，把：

LLM 流式输出时间
工具 I/O 等待时间
环境预热时间

尽可能重叠起来，降低：

首次工具结果返回延迟
多步推理链路总时长
高 I/O 工具的用户体感停顿
1.2 必须满足的系统不变量
不变量 A：关闭 speculation 后，正确性完全不变

推测执行只能加速，不能成为 correctness 前提。

不变量 B：Shadow 不等于 Commit

推测执行产生的是“候选结果”，不是正式业务提交。

不变量 C：最终裁决权属于 TransactionKernel

ShadowEngine 只能猜。
TransactionKernel 负责：

ADOPT
JOIN
CANCEL
REPLAY
不变量 D：所有推测任务必须可追踪

不得存在不可见的后台幽灵任务。

不变量 E：失败必须安全降级

任意 speculative failure 都必须能退化到普通同步工具执行路径。

1.3 非目标

本阶段不做：

任意写操作的 speculative commit
无限分支推测
跨会话长期 speculative cache
复杂自学习策略调参
模型输出内容层面的文本猜测执行
2. 总体架构
2.1 逻辑分层

将推测执行拆成 4 层：

Layer 1: Candidate Extraction

从流式 tool 参数中提取候选工具调用。

Layer 2: Speculation Decision

依据工具策略、参数稳定度、预算与系统压力决定是否启动 shadow task。

Layer 3: Shadow Execution

以受控方式执行影子任务，并注册到 Registry。

Layer 4: Authoritative Resolution

当 TransactionKernel 正式需要工具结果时，执行：

ADOPT
JOIN
CANCEL
REPLAY
2.2 在 Polaris 中的位置
LLM Stream
   ↓
turn_transaction_controller.execute_stream()
   ↓
StreamShadowEngine.consume_event()/consume_delta()
   ↓
CandidateToolCall + StabilityScorer
   ↓
SpeculationCoordinator
   ↓
ShadowTaskRegistry.start_shadow_task()
   ↓
ToolBatchRuntime.execute(..., speculative=True)
   ↓
TransactionKernel.resolve_or_execute()
   ↓
ADOPT / JOIN / CANCEL / REPLAY
2.3 事务语义原则
正式路径

正式工具调用仍由 TransactionKernel 驱动。

影子路径

Shadow task 只做“候选准备”，不会直接改变系统 authoritative state。

边界

ToolBatchRuntime 只是执行器。
是否采用其结果，必须由 TransactionKernel 决定。

5. 状态机定义
5.1 Candidate 状态机
INCOMPLETE
  ├─(json_closed)────────────→ SYNTACTIC_COMPLETE
  ├─(still_mutating)─────────→ INCOMPLETE
SYNTACTIC_COMPLETE
  ├─(schema_ok)──────────────→ SCHEMA_VALID
  ├─(schema_fail)────────────→ INCOMPLETE
SCHEMA_VALID
  ├─(stable_score_ok)────────→ SEMANTICALLY_STABLE
  ├─(field_mutated_again)────→ INCOMPLETE / SYNTACTIC_COMPLETE
SEMANTICALLY_STABLE
  ├─(minor_noncritical_change)→ SEMANTICALLY_STABLE
  ├─(critical_field_changed)──→ INCOMPLETE
  └─(stream_end)──────────────→ terminal candidate snapshot
升级条件
json_closed
schema_ok
critical fields unchanged for N deltas or T ms
end_tag_seen
回退条件
任意关键字段被覆写
tool_name 改变
关键字段 canonical hash 改变
5.2 ShadowTask 状态机
CREATED
  → ELIGIBLE
  → STARTING
  → RUNNING
  → COMPLETED
  → ADOPTED

异常路径：

RUNNING → FAILED
RUNNING → CANCEL_REQUESTED → CANCELLED
COMPLETED → EXPIRED
COMPLETED → ABANDONED
RUNNING → ABANDONED (salvage decision)

resolve_or_execute(spec_key)
  ├─ no task ───────────────→ REPLAY
  ├─ task completed match ─→ ADOPT
  ├─ task running match ───→ JOIN
  ├─ task mismatch ────────→ CANCEL old + REPLAY
  ├─ task failed ──────────→ REPLAY
  └─ task cancelled ───────→ REPLAY

TURN_ACTIVE
  ├─ refusal / safety block ─────→ TURN_ABORTING

class StreamShadowEngine:
    def __init__(
        self,
        decoder: CandidateDecoder,
        scorer: StabilityScorer,
        coordinator: "SpeculationCoordinator",
        resolver: "SpeculationResolver",
    ) -> None:
        self._decoder = decoder
        self._scorer = scorer
        self._coordinator = coordinator
        self._resolver = resolver

    # 兼容现有 turn_transaction_controller 的调用方式
    def consume_delta(self, delta: str) -> dict[str, Any] | None:
        """兼容现有接口：先缓冲，再尝试低置信度预测。"""
        if not delta:
            return None
        self._buffer.append(delta)
        # 现有代码：仅做关键词触发（保留兼容逻辑）
        # 升级后：调用 decoder 做增量解析
        combined = "".join(self._buffer)
        if "<tool_call>" in combined or "```tool" in combined:
            return {"trigger": "keyword", "confidence": 0.1}
        return None

    async def consume_stream_event(
        self,
        *,
        turn_id: str,
        stream_id: str,
        event_type: str,
        content: str | None,
        tool_name: str | None = None,
        tool_args: dict | None = None,
        call_id: str | None = None,
    ) -> None:
        # 1) 优先处理显式 tool_call 事件（对应现有 _try_speculate_tool_call 路径）
        if event_type == "tool_call" and tool_name is not None and tool_args is not None:
            await self._coordinator.speculate_from_explicit_tool_call(
                turn_id=turn_id,
                stream_id=stream_id,
                tool_name=tool_name,
                tool_args=tool_args,
                call_id=call_id,
            )
            return

        # 2) 对增量参数流做候选提取（对应现有 consume_delta 路径的升级）
        if event_type not in {"thinking_chunk", "content_chunk", "tool_args_delta"}:
            return

        if not content:
            return

        candidate_updates = self._decoder.consume_delta(
            turn_id=turn_id,
            stream_id=stream_id,
            delta=content,
        )

        for candidate in candidate_updates:
            scored = self._scorer.score(candidate)
            await self._coordinator.speculate_if_eligible(scored)

    async def resolve_or_execute(
        self,
        *,
        turn_id: str,
        call_id: str,
        tool_name: str,
        args: dict,
    ):
        return await self._resolver.resolve_or_execute(
            turn_id=turn_id,
            call_id=call_id,
            tool_name=tool_name,
            args=args,
        )
6.2 推测准入阶段
class SpeculationCoordinator:
    def __init__(
        self,
        policy_registry: "PolicyRegistry",
        budget: "SpeculationBudgetGovernor",
        registry: "ShadowTaskRegistry",
        runtime: "ToolBatchRuntimeAdapter",
        metrics: "SpeculationMetrics",
        fingerprint_builder: "FingerprintBuilder",
    ) -> None:
        self._policy_registry = policy_registry
        self._budget = budget
        self._registry = registry
        self._runtime = runtime
        self._metrics = metrics
        self._fingerprint_builder = fingerprint_builder

    async def speculate_if_eligible(self, candidate: CandidateToolCall) -> None:
        if candidate.tool_name is None:
            return

        policy = self._policy_registry.get(candidate.tool_name)
        if policy.speculate_mode == "forbid":
            return

        if candidate.parse_state != ParseState.SEMANTICALLY_STABLE:
            return

        if candidate.stability_score < policy.min_stability_score:
            return

        normalized_args = normalize_args(candidate.tool_name, candidate.partial_args)
        env_fingerprint = self._fingerprint_builder.build_env_fingerprint()
        spec_key = self._fingerprint_builder.build_spec_key(
            tool_name=candidate.tool_name,
            normalized_args=normalized_args,
            env_fingerprint=env_fingerprint,
        )

        if self._registry.exists_active(spec_key):
            return

        cost_estimate = estimate_tool_cost(candidate.tool_name, normalized_args)
        allowed = self._budget.allow(
            tool_name=candidate.tool_name,
            policy=policy,
            estimated_cost=cost_estimate,
            confidence=candidate.stability_score,
        )
        if not allowed:
            self._metrics.record_skip(candidate, reason="budget_denied")
            return

        await self._registry.start_shadow_task(
            turn_id=candidate.turn_id,
            candidate_id=candidate.candidate_id,
            tool_name=candidate.tool_name,
            normalized_args=normalized_args,
            spec_key=spec_key,
            env_fingerprint=env_fingerprint,
            policy=policy,
            runtime=self._runtime,
            cost_estimate=cost_estimate,
        )
        self._metrics.record_started(candidate, spec_key=spec_key)
6.3 正式执行阶段：四动作
class SpeculationResolver:
    def __init__(
        self,
        registry: "ShadowTaskRegistry",
        runtime: "ToolBatchRuntimeAdapter",
        salvage_policy: "SalvagePolicy",
        metrics: "SpeculationMetrics",
        fingerprint_builder: "FingerprintBuilder",
    ) -> None:
        self._registry = registry
        self._runtime = runtime
        self._salvage_policy = salvage_policy
        self._metrics = metrics
        self._fingerprint_builder = fingerprint_builder

    async def resolve_or_execute(
        self,
        *,
        turn_id: str,
        call_id: str,
        tool_name: str,
        args: dict,
    ):
        normalized_args = normalize_args(tool_name, args)
        env_fingerprint = self._fingerprint_builder.build_env_fingerprint()
        spec_key = self._fingerprint_builder.build_spec_key(
            tool_name=tool_name,
            normalized_args=normalized_args,
            env_fingerprint=env_fingerprint,
        )

        task = self._registry.lookup(spec_key)

        if task is None:
            self._metrics.record_replay(turn_id, call_id, tool_name, reason="miss")
            return await self._runtime.execute_authoritative(tool_name, normalized_args, call_id)

        if task.state == ShadowTaskState.COMPLETED:
            self._metrics.record_adopt(turn_id, call_id, tool_name, spec_key)
            return await self._registry.adopt(task.task_id, call_id)

        if task.state in {ShadowTaskState.STARTING, ShadowTaskState.RUNNING}:
            self._metrics.record_join(turn_id, call_id, tool_name, spec_key)
            return await self._registry.join(task.task_id, call_id)

        if task.state in {ShadowTaskState.FAILED, ShadowTaskState.CANCELLED, ShadowTaskState.EXPIRED}:
            self._metrics.record_replay(turn_id, call_id, tool_name, reason=task.state.value)
            return await self._runtime.execute_authoritative(tool_name, normalized_args, call_id)

        self._metrics.record_replay(turn_id, call_id, tool_name, reason="unexpected_state")
        return await self._runtime.execute_authoritative(tool_name, normalized_args, call_id)
7. ShadowTaskRegistry 详细设计
7.1 Registry 职责

Registry 必须是原子协调者，而不是简单 dict。

必须提供：

start_shadow_task
lookup
exists_active
adopt
join
cancel
abandon
expire
drain_turn

7.2 关键约束
同一 spec_key 同时最多一个 active task

防止重复 speculative execution。

adopt/join/cancel 必须具备锁保护

避免 race condition。

任务完成后不能无限滞留

必须过 TTL 自动过期。

7.3 伪代码
class ShadowTaskRegistry:
    def __init__(
        self,
        *,
        clock: "Clock",
        cancel_manager: "CancellationManager",
        cache: "EphemeralSpecCache",
        metrics: "SpeculationMetrics",
    ) -> None:
        self._clock = clock
        self._cancel_manager = cancel_manager
        self._cache = cache
        self._metrics = metrics
        self._tasks_by_id: dict[str, ShadowTaskRecord] = {}
        self._active_spec_index: dict[str, str] = {}
        self._turn_index: dict[str, set[str]] = {}
        self._lock = asyncio.Lock()

    async def start_shadow_task(
        self,
        *,
        turn_id: str,
        candidate_id: str,
        tool_name: str,
        normalized_args: dict,
        spec_key: str,
        env_fingerprint: str,
        policy: ToolSpecPolicy,
        runtime: "ToolBatchRuntimeAdapter",
        cost_estimate: float,
    ) -> ShadowTaskRecord:
        async with self._lock:
            existing = self._get_active_by_spec_key(spec_key)
            if existing:
                return existing

            task_id = new_id("shadow")
            cancel_token = self._cancel_manager.new_token(task_id=task_id, turn_id=turn_id)

            record = ShadowTaskRecord(
                task_id=task_id,
                origin_turn_id=turn_id,
                origin_candidate_id=candidate_id,
                tool_name=tool_name,
                normalized_args=normalized_args,
                spec_key=spec_key,
                env_fingerprint=env_fingerprint,
                policy_snapshot=policy,
                state=ShadowTaskState.STARTING,
                started_at=self._clock.monotonic(),
                cost_estimate=cost_estimate,
            )
            self._tasks_by_id[task_id] = record
            self._active_spec_index[spec_key] = task_id
            self._turn_index.setdefault(turn_id, set()).add(task_id)

            async def _runner():
                record.state = ShadowTaskState.RUNNING
                try:
                    result = await runtime.execute_speculative(
                        tool_name=tool_name,
                        args=normalized_args,
                        timeout_ms=policy.timeout_ms,
                        cancel_token=cancel_token,
                    )
                    record.result = result
                    record.state = ShadowTaskState.COMPLETED
                    record.finished_at = self._clock.monotonic()
                    record.expiry_at = record.finished_at + (policy.cache_ttl_ms / 1000.0)
                    await self._cache.put(record)
                    return result
                except asyncio.CancelledError:
                    record.state = ShadowTaskState.CANCELLED
                    record.finished_at = self._clock.monotonic()
                    raise
                except Exception as exc:
                    record.state = ShadowTaskState.FAILED
                    record.error = repr(exc)
                    record.finished_at = self._clock.monotonic()
                    raise

            record.future = asyncio.create_task(_runner(), name=f"shadow:{task_id}")
            return record

    def lookup(self, spec_key: str) -> ShadowTaskRecord | None:
        task_id = self._active_spec_index.get(spec_key)
        if not task_id:
            return None
        return self._tasks_by_id.get(task_id)

    async def adopt(self, task_id: str, call_id: str):
        async with self._lock:
            record = self._tasks_by_id[task_id]
            if record.state != ShadowTaskState.COMPLETED:
                raise RuntimeError("cannot adopt non-completed task")
            record.state = ShadowTaskState.ADOPTED
            record.adopted_by_call_id = call_id
            return record.result

    async def join(self, task_id: str, call_id: str):
        record = self._tasks_by_id[task_id]
        record.adopted_by_call_id = call_id
        return await record.future

    async def cancel(self, task_id: str, reason: str) -> None:
        async with self._lock:
            record = self._tasks_by_id[task_id]
            if record.state not in {ShadowTaskState.STARTING, ShadowTaskState.RUNNING}:
                return
            record.state = ShadowTaskState.CANCEL_REQUESTED
            record.cancel_reason = reason
            self._cancel_manager.cancel(task_id)
            if record.future:
                record.future.cancel()
8. 参数稳定度算法
8.1 原则
不是“JSON 闭合即稳定”

而是“关键字段在最近一段时间内不再变化”。

不同工具字段权重不同

例如：

repo_rg.query 权重大
repo_rg.top_k 权重小
read_file.path 权重极大
web_fetch.url 权重极大
8.2 评分公式建议
stability_score =
  schema_factor * 0.25
+ end_tag_factor * 0.15
+ critical_field_quietness * 0.35
+ overwrite_penalty_adjusted * 0.15
+ canonical_hash_consistency * 0.10

其中：

schema_factor: schema 是否通过
end_tag_factor: 是否看到明确结束标记
critical_field_quietness: 关键字段最近 N 次 delta 未变化
overwrite_penalty_adjusted: 有无关键字段覆写
canonical_hash_consistency: normalize 前后 hash 是否稳定
8.3 示例实现
class StabilityScorer:
    def __init__(self, schema_registry: "ToolSchemaRegistry", field_weights: "FieldWeightRegistry"):
        self._schema_registry = schema_registry
        self._field_weights = field_weights

    def score(self, candidate: CandidateToolCall) -> CandidateToolCall:
        weights = self._field_weights.get(candidate.tool_name or "")
        schema_ok = self._schema_registry.validate_partial(candidate.tool_name, candidate.partial_args)

        critical_quietness = self._compute_critical_quietness(candidate, weights)
        overwrite_penalty = self._compute_overwrite_penalty(candidate, weights)
        hash_consistency = self._compute_hash_consistency(candidate)

        score = 0.0
        score += 0.25 if schema_ok else 0.0
        score += 0.15 if candidate.end_tag_seen else 0.0
        score += 0.35 * critical_quietness
        score += 0.15 * max(0.0, 1.0 - overwrite_penalty)
        score += 0.10 * hash_consistency

        candidate.schema_valid = schema_ok
        candidate.stability_score = min(1.0, max(0.0, score))

        if schema_ok and candidate.parse_state == ParseState.SYNTACTIC_COMPLETE:
            candidate.parse_state = ParseState.SCHEMA_VALID

        if (
            candidate.parse_state == ParseState.SCHEMA_VALID
            and candidate.stability_score >= 0.82
            and critical_quietness >= 0.9
        ):
            candidate.parse_state = ParseState.SEMANTICALLY_STABLE

        return candidate

class SalvagePolicy:
    def decide(
        self,
        task: ShadowTaskRecord,
        *,
        likely_reuse: bool,
        remaining_cost_ratio: float,
        queue_pressure: float,
    ) -> SalvageDecision:

```python
class StreamShadowEngine:
    def __init__(self, speculative_executor: SpeculativeExecutor) -> None:
        self._speculative_executor = speculative_executor
        self._buffer: list[str] = []

    def consume_delta(self, delta: str) -> dict[str, Any] | None:
        if not delta:
            return None
        self._buffer.append(delta)
        combined = "".join(self._buffer)
        prediction = {"tool_name": None, "arguments": None, "confidence": 0.0}
        if "<tool_call>" in combined or "```tool" in combined:
            prediction["confidence"] = 0.1
        return prediction

    async def speculate_tool_call(
        self, *, tool_name: str, arguments: dict[str, Any], call_id: str
    ) -> dict[str, Any]:
        normalized = tool_name.strip().lower().replace("-", "_")
        if normalized not in ToolBatchRuntime.READONLY_TOOLS:
            return {"enabled": True, "result": None, "error": "non_readonly_tool"}
        invocation = ToolInvocation(
            call_id=ToolCallId(call_id or f"spec_{normalized}"),
            tool_name=tool_name,
            arguments=arguments or {},
            effect_type=ToolEffectType.READ,
            execution_mode=ToolExecutionMode.READONLY_PARALLEL,
        )
        return await self._speculative_executor.speculate(invocation)
```

A.2 speculative_executor.py（当前骨架）

```python
class SpeculativeExecutor:
    ENABLE_SPECULATIVE_EXECUTION: bool = False

    def __init__(self, batch_runtime: ToolBatchRuntime, *, enabled: bool | None = None) -> None:
        self._batch_runtime = batch_runtime
        self._enabled = is_speculative_execution_enabled() if enabled is None else bool(enabled)

    async def speculate(self, tool_invocation: ToolInvocation) -> dict[str, Any]:
        if not self._enabled:
            return {"enabled": False, "result": None, "error": "speculative_execution_disabled"}
        batch = ToolBatch(
            batch_id=BatchId("speculative-batch"),
            parallel_readonly=[tool_invocation],
        )
        try:
            receipts = await self._batch_runtime.execute_batch(batch)
            if receipts and receipts[0].results:
                result = receipts[0].results[0]
                return {
                    "enabled": True,
                    "result": result.result if result.status == "success" else None,
                    "error": result.status if result.status != "success" else None,
                }
            return {"enabled": True, "result": None, "error": "no_results"}
        except Exception as exc:
            return {"enabled": True, "result": None, "error": str(exc)}
```

A.4 _drain_speculative_tasks（当前粗暴取消逻辑）

```python
async def _drain_speculative_tasks(
    self,
    tasks: list[tuple[str, asyncio.Task[dict[str, Any]]]],
    *,
    ledger: TurnLedger | None = None,
    timeout_s: float = 0.2,
) -> None:
    if not tasks:
        return
    done, pending = await asyncio.wait(
        [t for _, t in tasks], timeout=timeout_s, return_when=asyncio.ALL_COMPLETED
    )
    for call_id, task in zip([c for c, _ in tasks], [t for _, t in tasks]):
        if task not in done:
            continue
        try:
            outcome = task.result()
        except Exception as exc:
            outcome = {"enabled": True, "result": None, "error": str(exc)}
        if isinstance(outcome, Mapping):
            ledger.record_speculative_outcome(call_id, outcome)
    for task in pending:
        task.cancel()
    if pending:
        await asyncio.gather(*pending, return_exceptions=True)
    logger.debug("speculative tasks drained: done=%s pending_cancelled=%s", len(done), len(pending))
```

@dataclass
class TurnLedger:
    turn_id: str
    started_at_ms: int = field(default_factory=lambda: int(time.time() * 1000))
    completed_at_ms: int = 0
    llm_calls: list[dict] = field(default_factory=list)
    tool_executions: list[dict] = field(default_factory=list)
    decisions: list[dict] = field(default_factory=list)
    tool_batch_count: int = 0
    events: list[TurnPhaseEvent] = field(default_factory=list)
    state_history: list[tuple[str, int]] = field(default_factory=list)
    # 以下字段已在当前代码中存在（第 142-146 行）
    kernel_guard_assert_count: int = 0
    kernel_guard_assert_failures: int = 0
    speculative_attempted_call_ids: set[str] = field(default_factory=set)
    speculative_successful_call_ids: set[str] = field(default_factory=set)
    canonical_tool_call_ids: set[str] = field(default_factory=set)

    def record_speculative_outcome(self, call_id: str, outcome: Mapping[str, Any]) -> None:
        if not call_id:
            return
        if bool(outcome.get("enabled")) is not True:
            return
        error = str(outcome.get("error") or "").strip()
        if error == "non_readonly_tool":
            return
        self.speculative_attempted_call_ids.add(call_id)
        if not error:
            self.speculative_successful_call_ids.add(call_id)

    def build_monitoring_metrics(self, final_kind: str) -> dict[str, float]:
        assert_count = max(0, self.kernel_guard_assert_count)
        assert_failures = max(0, self.kernel_guard_assert_failures)
        speculative_attempts = len(self.speculative_attempted_call_ids)
        speculative_false_positives = len(
            self.speculative_attempted_call_ids - self.canonical_tool_call_ids
        )
        speculative_hits = len(
            self.speculative_successful_call_ids & self.canonical_tool_call_ids
        )
        assert_fail_rate = (assert_failures / assert_count) if assert_count > 0 else 0.0
        hit_rate = (speculative_hits / speculative_attempts) if speculative_attempts > 0 else 0.0
        false_positive_rate = (
            speculative_false_positives / speculative_attempts if speculative_attempts > 0 else 0.0
        )
        return {
            "transaction_kernel.violation_count": float(assert_failures),
            "turn.single_batch_ratio": 1.0 if self.tool_batch_count <= 1 else 0.0,
            "workflow.handoff_rate": 1.0 if final_kind == "handoff_workflow" else 0.0,
            "kernel_guard.assert_fail_rate": float(assert_fail_rate),
            "speculative.hit_rate": float(hit_rate),
            "speculative.false_positive_rate": float(false_positive_rate),
        }
